# dat803course

## Requirements
- Python >= 3.14.*
- [PDM](https://pdm-project.org) — `pip install pdm`

## Setup

```bash
pdm install
```

## Run (quick guide)

**Jupyter Lab**
```bash
pdm run jupyter lab
```

**VS Code**
1. Open the `.ipynb` file
2. Click **Select Kernel** (top right)
3. Choose **Python Environments** → select the `.venv` created by PDM
